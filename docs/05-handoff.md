## 0. TL;DR — 이것만 하면 된다

**우주 머지-드롭 퍼즐 아케이드 "NOVA"를 HTML5로 만들어라.**
드래그로 조준해 천체를 떨어뜨리고, 같은 천체끼리 닿으면 합쳐져 더 큰 천체가 된다
(별먼지→…→블랙홀). 넘치면 게임오버, **광고 보고 부활**(런당 1회)해 이어간다.
강한 이펙트 + iPhone 햅틱 + 다크 네온 우주 아트. **웹으로 배포**해 플레이 링크를 만든다.

스펙은 `docs/02`, 물리·기술은 `docs/03`, 배포는 `docs/04`, 개발 규칙은 `CLAUDE.md`.

## 4. 고정 제약 (변경하려면 먼저 질문)

- **HTML5**, 패키지 < 50MB, 60fps (바디 100개+에서도).
- 플랫폼 기능(광고·햅틱·저장)은 **어댑터 계층** 경유. 게임 로직에서 SDK 직접 호출 금지.
- 게임은 **무한 점수형**, **원핑거**, **광고 부활로 계속** 가능해야 한다.
- 햅틱은 향상이지 필수가 아니다. 밸런스 수치는 전부 `config.ts`.
