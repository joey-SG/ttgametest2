# 02 · 게임 디자인 스펙

리서치 결론(`01-market-research.md`)을 하나의 구체적 게임으로 확정한다.
구현 담당(Sonnet 5)은 이 스펙을 "무엇을 만족해야 하는가"로 읽고, 구현 방식은 자유롭게 선택한다(Goal-Driven).

---

## 1. 게임 컨셉 (확정)

### 이름: **NOVA** (노바)

**한 문장**: 좌우 드래그로 조준해 떨어뜨린 천체가 물리로 굴러 쌓이고, 같은 천체끼리 닿으면
**합쳐져 더 큰 천체**가 된다(먼지→…→블랙홀). 우주 상자가 넘치면 끝.
광고를 보면 보드를 정리하고 이어간다.

**왜 이 컨셉인가 (설계 근거):**
- **2026 최강세 메커니즘**: 머지 + 물리 드롭 (docs/01 §1).
- **검증된 코어**: Suika Game으로 코어 루프 검증 완료. 근접 실패·연쇄 쾌감·차오르는 압박.
- **원핑거·3초 이해**: 드래그로 좌우 조준, 손 떼면 드롭. 튜토리얼 불필요.
- **juice 친화**: "합쳐지는 순간"에 파티클·글로우·펀치·햅틱을 몰아줄 수 있음.
- **테마 차별화**: 과일(클론 낙인) 대신 **우주/천체** — 다크 배경 + 네온 글로우가
  테마와 자연 결합, TikTok 다크 피드에서 시인성↑. 블랙홀 엔드티어는 클립 후킹.
- **#1(PULSE)과 완전 차별**: 타이밍/리듬 → 공간 배치/물리 예측.

## 3. 코어 루프 (반드시 구현)

```
[타이틀: 탭하여 시작]
        │
        ▼
[플레이] ── 드래그(좌우 조준) → 릴리즈(드롭) → 물리 낙하·충돌·안착
        │         같은 티어 접촉 → 머지! 점수+, 이펙트+햅틱
        │         연쇄 머지(체인) → 콤보 배율 ×2/×3/×4
        │
        └─ 오버플로우 라인 위에 천체가 일정 시간 머무름
                          ▼ (경고 점멸 후)
                   [게임 오버 화면]
                    ├─ "다시하기" (0초 재시작)
                    ├─ "광고 보고 부활" → 보드 정리 후 이어하기 (런당 1회)
                    ├─ "광고 보고 점수 2배" (런당 1회)
                    └─ "공유" (최고 천체 + 점수)
```

**불변 규칙:**
- 재시작은 **0초**. 로딩·모달 대기 없음.
- 게임오버는 처벌이 아니라 **성과 요약**("지구까지 만들었다!")과 재도전 초대.

## 4. 판정·배율 규칙 (확정)

- **머지 판정**: 같은 티어 두 천체가 접촉하면 접점에서 다음 티어로 합체. 같은 프레임
  다중 충돌은 한 쌍만 처리(중복 머지 방지 쿨다운).
- **체인 콤보**: 머지 후 **1.5초 내** 연이은 머지 = 체인. 체인 2/3/4+ 단계에서 배율 ×2/×3/×4.
  체인이 끊기면 배율 ×1로 리셋. (Beatstar식 배율 사다리의 물리 버전 — #1에서 검증한 철학 계승)
- **게임오버 판정**: 오버플로우 라인(컨테이너 상단) 위에 천체 중심이 **2.0초** 연속 체류 시.
  체류 중 경고(라인 점멸 + 위험 햅틱). 드롭 직후의 일시적 튐은 게임오버 아님.
- **부활**: 광고 완주 → **하위 3티어 전체 제거 + 라인 위 천체 제거** → 점수·보드 유지, 계속.
  **런당 1회**, 스킵하면 횟수 미소모 (#1 Subway Surfers 원칙 계승).

## 5. 수익화 & 무한 지속 (보상형 광고)

- **Watch to Revive (부활)**: 게임오버 시 광고 완주 → 보드 정리 후 이어하기. 런당 1회.
- **Double Score (2배)**: 게임오버 화면에서 광고 완주 → 최종 점수 2배. 런당 1회, 사용 후 버튼 숨김.
- 광고 SDK는 어댑터 뒤로 추상화 (`docs/03`). 콜백 boolean(완주 여부)으로만 보상 지급.
- 웹 데모 폴백: 3초 카운트다운 오버레이 (#1 검증 패턴 재사용).

## 6. 아트 디렉션 — "우주 네온, 깔끔·심플"

- **다크 스페이스 배경**(거의 검정 + 미세한 별) + 천체는 **절차적 그라디언트 원 + 네온 글로우**.
- 천체에 **미니멀 표정**(점 눈 + 입 커브) — Suika의 "귀여움" 요인 계승, 절차 드로잉으로 용량 0.
- 토성 고리·태양 코로나·블랙홀 강착원반 등 티어 시그니처는 단순 도형으로.
- 폰트 1개, UI 최소. 이미지 에셋 0 (전부 절차 드로잉) — 50MB 제한 여유 확보.
- 세로 화면 기준, 노치/세이프에어리어 대응.

## 7. 이펙트(juice) 체크리스트

- [ ] 머지: 접점 파티클 버스트 + 새 천체 스케일 펀치(팝) + 글로우 플래시.
- [ ] 체인: 단계 상승마다 색 변화·화면 가장자리 글로우·사운드 피치 상승.
- [ ] 고티어 머지(8+): 화면 셰이크 + 강한 햅틱 + 링 웨이브.
- [ ] 빅뱅(블랙홀 머지): 전용 대형 이펙트 (화이트아웃 + 충격파).
- [ ] 오버플로우 경고: 라인 점멸 + 위험 햅틱 + 심장박동 펄스.
- [ ] 드롭: 가벼운 낙하 트레일. 안착 시 미세 스쿼시.
- [ ] 모든 이펙트는 60fps 유지 선에서 (파티클 풀링).

## 8. 사운드

- 절차 생성(WebAudio) 우선: 머지 팝(티어 높을수록 낮고 풍성), 체인 피치 상승, 게임오버 붐.
- 음소거 시청 대비: **햅틱+비주얼만으로** 손맛 완성 (#1 원칙 계승).

## 9. 햅틱 매핑

| 순간 | 종류 |
|---|---|
| 드롭 | select (가벼움) |
| 머지 | success |
| 체인 3+/고티어 머지 | combo (강함) |

## 10. 화면 목록 (최소)

1. **타이틀** — 로고 + "탭하여 시작" + 최고점수·최고 천체.
2. **플레이** — 보드 + 점수 + 다음 천체 미리보기 + 진화 사다리(작게, 측면).
3. **게임오버** — 점수, 최고점수 갱신 강조, 최고 도달 천체, [다시][광고 부활][2배][공유].
4. (첫 판 한정) 드롭 힌트 1줄 오버레이 — 첫 머지 성공 시 소멸.
